This is a simple app to handle a few different calculations for students. Firstly, I can calculate the score needed on a final exam in order to acheive a desired grade in the course. Along with that, there is GPA calculation function that allows the user to enter the credit hours and grade for each course and it returns the GPA for that term.

The grading scale used is on the +/- scale:

  • A+   4.0
  • A     4.0
  • A-    3.7
  • B+   3.3
  • B     3.0
  • B-   2.7
  • C+   2.3
  • C     2.0
  • C-   1.7
  • D+   1.3
  • D     1.0
  • D-   0.7
  • F     0.0
